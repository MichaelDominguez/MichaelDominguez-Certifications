# M.D. Certifications

This repository showcases my professional certifications and continued development across cybersecurity, cloud platforms, infrastructure, and technical systems. The certificates are organized into two focused categories so recruiters and hiring managers can review the training most relevant to technical and security-focused roles.

## Cybersecurity Certifications

This folder contains certifications that directly support cybersecurity readiness, security operations knowledge, platform security concepts, identity and access topics, and vendor-based security training. The Google and Microsoft certificates are grouped here because they are the most relevant to cybersecurity career preparation and technical security fundamentals.

## Cloud, Infrastructure, and Systems Certifications

This folder contains supporting technical certificates connected to governance, compliance, systems awareness, technical foundations, and platform-based learning. These certificates show broader technical development beyond one specific security tool or course.

## Purpose

The goal of this repository is to present certifications in a clean, professional format rather than leaving them as an unorganized collection of files. Each document supports my ongoing preparation for cybersecurity, information technology, and security-focused technical roles.
